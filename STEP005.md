# PHP基礎 05
## ジャンケンゲームVer.0の作成
ジャンケンゲームを作成します。  
これまで学習してきたことを使って作成します。  

## 作成手順
1. テンプレート(HTML)の作成  
    1. プレイヤーの手を選択するページ(gamestart.html)  
    1. 結果を表示するページ(result.php)  
1. PHPプログラムの作成  
    1. 結果ページ(result.php)にPHPブロックを追加  
    1. 送信されて来たPOSTデータを取得  
    1. コンピューターの手を乱数で決定  
    1. 勝敗を決定  
    1. 画面に表示  
作成例はSAMPLE03にあります。(10月26日現在)

## ヒント
勝敗はジャンケンアルゴリズムを使用する  
|  手 |  値  |
| ---- | ---- |
|  グー  |  0  |
|  チョキ  |  1  |
|  パー  |  2  |
と数字を置き換えて  
(自分の手 - 相手の手 + 3) % 3
|  結果 |  勝敗  |
| ---- | ---- |
|  0  |  ドロー  |
|  1  |  負  |
|  2  |  勝  |
となります。
